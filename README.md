# Essay Question Generator

---

[Link to the site](https://josie5734.github.io/EssayQuestionGenerator/)

This site randomly generates examples Question 6 Essay Question from Edexcel's A level Music Exam

I made this to assist me in revising for this question as there are a limited number of existing questions from the past papers

---

Click generate to generate a question

Each subject, artist and song has a checkbox, which removes that subject/artist/song from being used in a generated question
